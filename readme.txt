Bangladeshi Payments Mobile is a custom WooCommerce payment gateway plugin for integrating bKash, Nagad, Rocket, and Upay in Bangladesh.

== Description ==
**Bangladeshi Payments Mobile** is a WooCommerce payment gateway plugin that integrates the most popular mobile payment services in Bangladesh, including **bKash, Nagad, Rocket, and Upay**. It allows customers to securely and efficiently make payments using their mobile accounts. Merchants benefit from a user-friendly interface, simple setup, and reliable payment processing.

### Key Features
- **Secure Integration**: Supports **bKash, Nagad, Rocket, and Upay** for secure mobile transactions.
- **Custom Checkout Fields**: Collect **Mobile Number** and **Transaction ID** during checkout for easy tracking.
- **Dynamic Transaction Fees**: Automatically calculates service charges based on the selected payment method.
- **Detailed Order Information**: View mobile payment details (number and transaction ID) in WooCommerce order admin.
- **Payment Method Management**: Activate or deactivate specific payment gateways (bKash, Nagad, Rocket, Upay) from a dedicated settings page.
- **Transaction Info Dashboard**: Access a detailed list of all transactions with Mobile Number, Transaction ID, and Payment Method for easy management.
- **Flexible Payment Gateway Options**: Enable or disable specific gateways based on your preferences.
- **Merchant Customization**: Edit titles, descriptions, and transaction charges for each payment method in the settings.
- **Simple Setup**: Configure all options from the WooCommerce settings page.
- **No Hidden Costs**: Completely free to use with no hidden fees or subscription costs.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/bangladeshi-payments-mobile/` directory or install it directly from the WordPress plugin screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to **WooCommerce > Settings > Payments**, and enable **Bangladeshi Payments Mobile**.
4. Configure the plugin settings with your **bKash**, **Nagad**, **Rocket**, or **Upay** account details.

== Frequently Asked Questions ==
= What payment methods does Bangladeshi Payments Mobile support? =
The plugin supports **bKash**, **Nagad**, **Rocket**, and **Upay**, popular mobile payment services in Bangladesh.

= Do I need accounts with these payment providers to use the plugin? =
Yes, you need active merchant accounts with **bKash**, **Nagad**, **Rocket**, or **Upay** to receive payments.

= Can customers use any mobile payment provider listed? =
Yes, customers can choose their preferred provider (bKash, Nagad, Rocket, or Upay) during checkout.

= Does the plugin calculate transaction fees? =
Yes, the plugin can calculate transaction charges during checkout based on the selected payment method.

= How do I configure Bangladeshi Payments Mobile? =
After activating the plugin, go to **WooCommerce > Settings > Payments** and configure your payment options.

= Is this plugin secure for processing mobile payments? =
Yes, the plugin follows secure practices for processing payment details. For added security, use SSL on your WooCommerce store.

= Can I customize the payment gateway title and description? =
Yes, you can customize the title and description for each payment method in the WooCommerce settings.

= Will I see payment details in WooCommerce orders? =
Yes, the mobile number and transaction ID will appear in the order details for easy tracking.

= Is this plugin compatible with WooCommerce Subscriptions or Pre-orders? =
Currently, the plugin does not support WooCommerce Subscriptions or Pre-orders. It is designed for one-time payments.
