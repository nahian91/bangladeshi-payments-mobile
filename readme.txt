Accept Mobile Payments in Bangladesh – WooCommerce Gateway for **bKash, Nagad, Rocket & Upay** with QR Code & Transaction Reports.

== Description ==
**Bangladeshi Payments Mobile – QR Code & Transaction Reports** is a powerful WooCommerce payment gateway plugin designed for **Bangladeshi merchants**. It allows you to accept payments via the most popular mobile financial services: **bKash, Nagad, Rocket, and Upay**.  

With QR code support and detailed transaction reports, this plugin makes mobile payments secure, fast, and easy to manage.

### Key Features
- **Secure Mobile Payments**: Accept **bKash, Nagad, Rocket, and Upay** securely on your WooCommerce store.
- **QR Code Payment Support**: Customers can pay by scanning a QR code for fast and error-free transactions.
- **Custom Checkout Fields**: Collect **Mobile Number** and **Transaction ID** for accurate payment tracking.
- **Dynamic Transaction Fees**: Automatically calculate service charges based on the selected payment method.
- **Transaction Reports Dashboard**: View a complete list of all transactions with Mobile Number, Transaction ID, Payment Method, and Order Status.
- **Flexible Payment Options**: Enable or disable specific gateways according to your preferences.
- **Admin Customization**: Edit gateway titles, descriptions, and transaction charges from WooCommerce settings.
- **Detailed Order Information**: Easily track mobile payments in WooCommerce order admin.
- **Simple Setup**: Configure everything directly from the WooCommerce settings page.
- **Free & Transparent**: No hidden fees or subscriptions.

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/bangladeshi-payments-mobile/` or install via the WordPress plugin screen.
2. Activate the plugin through the 'Plugins' screen.
3. Go to **WooCommerce > Settings > Payments**, enable **Bangladeshi Payments Mobile – QR Code & Transaction Reports**.
4. Configure the plugin settings with your **bKash**, **Nagad**, **Rocket**, or **Upay** account details.

== Frequently Asked Questions ==
= Which payment methods are supported? =
Supports **bKash**, **Nagad**, **Rocket**, and **Upay**.

= Do I need merchant accounts? =
Yes, active accounts with the respective providers are required.

= Can customers choose their preferred provider? =
Yes, they can select **bKash, Nagad, Rocket, or Upay** during checkout.

= Are transaction fees calculated automatically? =
Yes, the plugin calculates fees dynamically per payment method.

= How do I set up the plugin? =
Activate it, then go to **WooCommerce > Settings > Payments** and configure your gateways.

= Is it secure for processing payments? =
Yes, it uses secure practices; SSL is recommended.

= Can I customize gateway titles and descriptions? =
Yes, fully customizable via settings.

= Will payment details appear in WooCommerce orders? =
Yes, mobile number and transaction ID are stored for easy tracking.

= Are QR codes supported? =
Yes, customers can scan QR codes to pay quickly and accurately.

= Can I view transaction reports? =
Yes, access a detailed dashboard listing all transactions, mobile numbers, transaction IDs, and payment methods.

= Does it support WooCommerce Subscriptions? =
Not currently, only one-time payments are supported.
